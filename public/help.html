Grocery List Help

This is an online, configurable grocery list you can use to keep track of what items you need to purchase.

View List

Use this page to view the items you need to purchase and their quantities. A checkbox is provided for each item to allow you to check them off as you get them. (Note that if you close down the page and then reopen it, all the checkboxes will be cleared out) You can also print this list.

Add & Remove Items

Use this page to specify the quantities you need for the items on your list. If you don’t need a particular item on your list at this time, set the quantity to zero.

Manage List

Use this page to create your list and manage the categories and groceries on it. You can create new categories, add grocery items to existing categories, remove grocery items, and remove categories.  Note that if you remove a category, all grocery items in that category are also removed.

You can also use this page to restart the entire grocery list.
